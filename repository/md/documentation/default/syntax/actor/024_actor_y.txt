# Actor Y
<!--

@description:

Returns the Y position of an actor defined by the `Y` property of the `Actor` command.

## Syntax
>A =  **Actor Y**( actor_index )

## Parameters
|**Parameter**|**Type**|**Description**|
|---|---|---|
|**`Index`**|index|Index of the Actor.|
