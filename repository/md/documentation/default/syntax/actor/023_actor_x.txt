Returns the X position of an actor defined by the `X` property of the `Actor` command.

## Syntax
>A =  **Actor X**( actor_index )

## Parameters
|**Parameter**|**Type**|**Description**|
|---|---|---|
|**`Index`**|index|Index of the Actor.|
