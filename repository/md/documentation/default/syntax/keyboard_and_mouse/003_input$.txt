X\$ **=Input$\(** numChars **)**

This function accepts a specified number of characters as input from the keyboard, and returns the result as a string.  It only accepts printable characters as input.  Special symbols may be entered using the Alt key.  

No visible output is shown.  The input goes directly into the function result.  No editing capabilities are provided.

## Parameters

| Parameter | Description |
|---|---|
| **numChars** | Number of characters to accept |
