fill memory block with the contents of a variable.
## Parameters

- **start**: The address to change. It should contain the "maginal number" returned by the Start function.
- **finish**: The address of the end of the area to fill. It should contain the "maginal number" returned by the Start function.
- **pattern**: The four-byte pattern that will be repeated when filling
