Turns off the specified LED(s) on the LED strip or ring connected to the specified port.
## Parameters

- **number**: The number of the LED in the strip to turn off (1 to 36, 0 for all LEDs)
- **port**: The number of the port where the strip is connected, 1, 2, or 0 for all ports
