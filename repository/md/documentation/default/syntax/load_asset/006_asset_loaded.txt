# Asset Loaded
<!--

Return True if an asset has been loaded with the "Load Asset" command, else False
## Parameters

- **index**: Index of the asset
- **type&dollar;**: Type of asset (may be "image", "module", "audio", "video", "spritesheet", "3d", "map")
