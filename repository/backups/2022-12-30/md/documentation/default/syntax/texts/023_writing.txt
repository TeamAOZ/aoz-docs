Define the logical operation done during the display of characters by the "Print" statement
## Parameters

- **mode1**: An Integer whose value indicates the writing mode.   0=REPLACE, 1=OR, 2=XOR, 3=AND, 4=IGNORE.
- **mode2**: An Integer whose value indicates the optional second writing mode.  0=Normal = Print Text and Background, 1=Paper = Only change the background (paper) color.
