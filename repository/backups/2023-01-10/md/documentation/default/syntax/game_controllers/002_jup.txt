**jUp(** joystick,lock **)** Returns true if the joystick is currently in the up position.
## Parameters
| **Parameter** | **Description** |
|:--------------|:----------------|
| **joystick**  | The number of the joystick |
| **lock** (optional)     | If true, the function will wait for the button to be released before being true again.  Can be used for a "single shot" mode. |
