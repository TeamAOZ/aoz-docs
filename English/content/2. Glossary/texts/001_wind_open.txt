This instruction opens a new Print window associated with the current screen.
## Parameters

|Parameter|Description|
|---|---|
| **index** | The index (ID) of Print window to open |
| **x** | The horizontal Print coordinate of the left border of the window in the screen |
| **y** | The vertical Print coordinate of the left border of the window in the screen |
| **width** | The width in characters of the new Print window |
| **height** | The width in characters of the new Print window |
| *border* | The number of the border character to use (optional) |
