# THE PROJECT FOLDERS
The Project Folders are all neatly stacked on the left side of the Edit Screen. Inside each folder is a whole host of sub-folders, and inside each sub-folder is a whole host of magic.

![enter image description here](https://doc.aoz.studio/assets/images/en/image053.png)
You can move, delete, rename and copy. Let's look at each folder in turn.

<img valign="middle" src="https://doc.aoz.studio/assets/images/en/image054.png" width="56px" />&nbsp;&nbsp;&nbsp;All your Applications are kept in this folder by default, with whatever names you have given them. Each Application folder can contain a load of other folders for Resources like images, icons, music and sound effects.

<img valign="middle" src="https://doc.aoz.studio/assets/images/en/image055.png" width="56px" />&nbsp;&nbsp;&nbsp;The Reference Manuals folder is where you can find this User guide, and old hands will find familiar items.

<img valign="middle" src="https://doc.aoz.studio/assets/images/en/image056.png" width="56px" />&nbsp;&nbsp;&nbsp;Tutorials are kept in this folder. They include a whole series of lessons about AOZ Studio along with learning via a series of classic entertainments.

<img valign="middle" src="https://doc.aoz.studio/assets/images/en/image057.png" width="56px" />&nbsp;&nbsp;&nbsp;The Demos folder contains classic games and special effects for you to take apart, adapt and put back together again as you like.

<img valign="middle" src="https://doc.aoz.studio/assets/images/en/image058.png" width="56px" />&nbsp;&nbsp;&nbsp;Games is where you will find a load of familiar and not so familiar ready-mades to enjoy, adapt and do what you want with.

<img valign="middle" src="https://doc.aoz.studio/assets/images/en/image059.png" width="56px" />&nbsp;&nbsp;&nbsp;Accessories/Utilities is the folder for useful links and resources like a font viewer, joystick and scancode testers.

**INTERACTIVE MANUAL AND HELP**
The Interactive Manual is behind the **?** Documentation Icon on the top right of the screen, or when you ask in the main **Search Bar** which is very useful.
