# Sprite Alpha
<!--

Set the global transparency of a sprite, also called "Alpha"
## Parameters

- **index**: Index of the sprite in the list of active sprites
- **alpha#**: Value of Alpha, from 1 (fully opaque) to 0 (fully transparent)
