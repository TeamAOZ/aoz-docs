Lights up the LEDs on the LED strip or ring connected to the specified port in the specified color(s).
## Parameters

- **r**: The RED value of the color to set (0 - 255)
- **g**: The GREEN value of the color to set (0 - 255)
- **b**: The BLUE value of the color to set (0 - 255)
- **number**: The number of the LED to affect, from 1 to 36. 0 indicating all the LEDs at once
- **port**: The number of the port where the strip is connected, 1, 2, or 0 for all ports
