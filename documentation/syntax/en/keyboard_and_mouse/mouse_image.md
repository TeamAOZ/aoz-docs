# Mouse Image

Change the mouse pointer by an image
- **URL&dollar;**: URL of the image to use. The image used must not be larger than 128x128 pixels. Supported formats are CUR, GIF, SVG and PNG. If this argument is an empty string, the mouse pointer will be reinitialized.
