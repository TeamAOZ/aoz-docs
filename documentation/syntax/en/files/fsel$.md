Open a file selector and return the name of the selected file
- **path&dollar;**: Path to the directory to display at start
- _default&dollar;_ (optional): Name of the default file to display as selected
- _title1&dollar;_ (optional): String to display as title of the selector
- _title2&dollar;_ (optional): String to display as secondary title of the selector
- _type&dollar;_ (optional): If "load", will display a LOAD file selector, if "Save" will display a SAVE file selector
