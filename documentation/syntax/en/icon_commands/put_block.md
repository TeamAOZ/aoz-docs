Draw a previously captured block in the current screen.
- **index**: Index of the block to draw
- _x_ (optional): Horizontal coordinate of the top-left corner of the block in the screen
- _y_ (optional): Vertical coordinate of the top-left corner of of the block in the screen
- _bitplanes_ (optional): Only in Amiga mode, defines the bitplanes to draw
- _billterMode_ (optional): Has no effect in AOZ.
