# Set Block Alpha
<table><tr><td colspan="2"><b>Compatibility</b></td></tr><tr><td><img src="https://drive.google.com/uc?export=view&id=1NbXQFq8_hw18wZSmQiAaH8PEkx0iN0ue" valign="center" all="AOZ" title="AOZ" /></td></tr></table>

Set the transparency alpha of a block, that will be applied the next time the block is drawn in one screen
- **index**: Index of the block to affect
- **alpha**: Value fo the transparency, from 0 to 1, 0= fully transparent, 1= fully opaque
