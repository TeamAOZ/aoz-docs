# Text Mode
<table><tr><td colspan="2"><b>Compatibility</b></td></tr><tr><td><img src="https://drive.google.com/uc?export=view&id=1NbXQFq8_hw18wZSmQiAaH8PEkx0iN0ue" valign="center" all="AOZ" title="AOZ" /></td></tr></table>

Sets the Text drawing mode to draw either solid colour text, outline text or bordered text. Text Mode is "fill" by default.
- **Mode&dollar;**: The Text drawing mode, can be either "fill", "outline" or "border". If this parameter is omitted, the mode will defalt to 'fill'.
