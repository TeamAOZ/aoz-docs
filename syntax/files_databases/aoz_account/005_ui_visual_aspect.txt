Choose the visual aspect for your `Account Authenticate` UI calls.

## Description

The authentication can take different visual aspects : choose it here. \
If you don't call this, a default "modern" visual aspect will be used. \
Visual aspects are `aoz`, `flat`, `modern`, `pictos`, `yellow`.

## Example

```basic
UI Visual Aspect "modern"
Account Authenticate UseToken = FALSE
```

## Notes

This visual aspect choice is used for the `Account Authenticate` instruction only, not for other `UI` instructions.
