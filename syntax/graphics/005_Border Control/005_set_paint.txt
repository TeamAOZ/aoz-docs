## Parameters

- **onOff**: Set to true or 1 for on, false or 0 for off.
---
## Description
When Set Paint is activated, all subsequent shapes drawn will have a outline border, drawn using the current line settings and the current Border Ink colour, Border Pattern or Border Fill Image.
