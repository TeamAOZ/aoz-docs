# Border Gradient Off

Turn off border gradients

## Syntax
**Border Gradient Off**
<div class="compatibility"><span>Compatibility  </span><img src="https://doc.aoz.studio/assets/images/common/compatibility/aoz_logo.png" valign="center" alt="AOZ" title="AOZ" /></div>

## Description
Turns off border gradient fill and return any subsequent borders to the previous fill setting.
