**FUNCTION** - Return the RGB value of a colour from the palette of the current screen
## Syntax

**= Colour (** _index_ **)**

## Parameters

- **index**: The index number of the colour in the palette of the current screen.

## Description

Use **Colour( )** to find out the current value of any colour in the palette of the current screen.
