**INSTRUCTION** - Set the RGB value of a colour in the palette the current screen

## Syntax
**Colour** *index, RGB*

## Parameters


- **index**: The index number of the colour in the palette of the current screen.
- **RGB**: The RGB colour value of the new colour.

## Description

**Colour** allows you to change any of the current screen|;s palette colour to any other 24 bit RGB colour. Once changed, all subsequent drawing operations using that **Ink** colour will have the new colour applied.

Whilst **Colour** can accept any number format, it|;s easiest to use the common hexadecimal format like this: $RRGGBB, for example $A055FF.

See also **Set Alpha** and **Shade**.
