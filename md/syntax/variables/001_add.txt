Fast Integer Addition
## Usage Options
**Add** variable,expression
**Add** variable,expression, base **To** top

## Parameters
**variable** The variable to add to
**expression** The value to add
**base** Starting Value
**top** Ending Value
## Description
- Using the simple form of of this instruction, **expression** is added to **variable**.

- Using the second form, if **variable**+**expression** exceeds **top**, then variable is set to **base**, otherwise **expression** is added to **variable**.
