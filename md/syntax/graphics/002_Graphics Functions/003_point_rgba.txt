## Syntax

~~**= Point RGBA**~~
~~**= Point RGBA ( x, y )**~~

~~Return the true colour RGBA value of a point within the current screen~~
## Parameters


- **x**: The horizontal coordinate of the point to test
- **y**: The vertical coordinate of the point to test
---
## Description
~~**Point RGBA** returns the 32 bit true colour value of the colour at the point of the current graphics coordinates.~~

~~**Point RGBA (** *x, y* **)** returns the 32 bit true colour value of the colour at the point of the coordinates *x, y.*~~
 
 ~~The 32 bit true colour value includes the Alpha channel which gives the colour transparency.~~

~~If the coordinates are outside of the screen boundary, the function will return -1.~~
