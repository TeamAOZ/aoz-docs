Updates the rendering of a 3D Screen.

#### Syntax
```basic
Load Asset "my_scene.aoz3d", "my_scene"
Td Init : Td Start
Screen 3D Open "my_screen", SceneId$="my_scene"
Do
	Td Screen Update "my_screen"
	Refresh
Loop
```

## Parameters
- **Index**: Number or name of the 3D screen.
